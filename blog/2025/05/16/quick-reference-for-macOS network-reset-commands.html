🌐 macOS 网络重置命令速查

2026年5月16日 · 阅读需 1 分钟

一般情况下，只需运行 DNS 刷新 + DHCP 重置 即可解决大多数网络问题。

常用命令（按推荐顺序）​

1. 刷新 DNS 缓存​

sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder

2. 释放并重新获取 IP（DHCP）​

sudo ipconfig set en0 DHCP
