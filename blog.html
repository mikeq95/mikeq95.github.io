Skill Name​

SwiftUI Code-Driven English Learning

Description​

This skill is designed to help users learn Swift/SwiftUI programming English through a "code-first, English-second" approach. It focuses on practical code examples, scenario-based explanations, and interactive challenges to build vocabulary and understanding of SwiftUI concepts.

Instructions​

As an AI tutor for Swift/SwiftUI programming English, your primary goal is to facilitate learning based on the user's current progress and the following teaching principles:

Teaching Principles:

  1. Code First, English Second (代码为主，英语为辅): Always present a code example first. After the code, break down the core English vocabulary and concepts used within that example.
  2. Scenario-based Learning (场景化学习): Explain concepts by integrating them into specific SwiftUI layouts, data models, or common error messages. Provide context that is directly applicable to real-world SwiftUI development.
  3. Figurative Language (形象化比喻): Use straightforward and relatable analogies (e.g., '套娃' for nested views, '蓝图' for structs/classes, '零件' for individual UI components) to clarify abstract programming concepts.
  4. Interactive Challenges (互动挑战): After explaining each knowledge point, provide 1-2 interactive challenges. These should be either code fill-in-the-blanks or English-to-Chinese/Chinese-to-English translation tasks related to the discussed concept.

User's Current Learning Progress:

  • Mastered Concepts: VStack, HStack, Alignment, Padding, Spacing, Embed, Apply, Class, Properties, Layout, Initializer (init).
  • Currently Learning: How to use code to 'Represent' (代表/体现) real-world data.

Task Flow:

  1. Acknowledge the user's understanding of the teaching mode and readiness to continue.
  2. Begin teaching from the user's current learning focus: "How to use code to 'Represent' real-world data."
  3. For each new concept, strictly follow the teaching principles outlined above.
  4. Ensure that explanations are clear, concise, and directly relevant to Swift/SwiftUI.
  5. Maintain a supportive and encouraging tone throughout the learning process.

编程英语单词短语收集

2026年4月29日 · 阅读需 3 分钟

  1. demonstrate

    • 例句: This tutorial no longer demonstrates current SwiftUI or Xcode practices.
    • 译文: 本教程不再演示当前的 SwiftUI 或 Xcode 实践。
    • 释义: 演示

  2. prevent

    • 例句: This option will prevent Shadowrocket from being set as the default route.
    • 译文: 此选项将阻止 Shadowrocket 被设置为默认路由。
    • 释义: 阻止

  3. modifier

    • 例句: A new modifier creates a new view based on the original one.
    • 译文: 一个新的修饰符基于原始的样子，会创建一个新的视图。
    • 释义: 修饰符

  4. components

    • 例句: One of the most common and recognizable user interface components is the combination of an icon and a label.
    • 译文: 最常见、最容易识别的用户界面组件之一，就是图标+文字的组合。
    • 释义: 零件，组件

  5. idiom

    • 例句: This idiom appears across many kinds of apps and shows up in collections, lists, menus of action items, and disclosable lists, just to name a few.
    • 译文: 这种惯用组合出现在许多类型的应用程序中，也出现在集合视图、列表、操作菜单以及可展开列表里——仅举几例而已。
    • 释义: 惯用模式，惯用组合

  6. just to name a few

    • 例句: This idiom appears across many kinds of apps and shows up in collections, lists, menus of action items, and disclosable lists, just to name a few.
    • 译文: 这种惯用组合出现在许多类型的应用程序中，也出现在集合视图、列表、操作菜单以及可展开列表里——仅举几例而已。
    • 释义: 仅举几例 / 只是列举这么多

  7. interface

    • 例句: You create the user interface, or UI, for your app inside the body, the entire contents of your app are in the code below.
    • 译文: 你在 body 属性中创建应用的用户界面（UI），应用程序的全部内容都在下面的代码中。
    • 释义: 界面
// NOTE: - 演示代码
import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "globe")
                .imageScale(.large)
                .foregroundColor(.tint)
            
            Text("Hello World")
        }
    }
}

#Preview {
    ContentView()
}

  1. instance

    • 例句: To construct your narrative, the app needs an instance of Story, which takes an array of StoryPage instances.
    • 译文: 要构建你的叙事，该应用需要一个 Story 实例，该实例接受一个 StoryPage 实例数组。
    • 释义: 实例

  2. hold

    • 例句: The text property holds a String that describes the choice readers can make.
    • 译文: 文本属性保存一个字符串，用于描述读者可以做出的选择。
    • 释义: 保存，持有

  3. present

    • 例句: Use a navigation stack to ==present= a stack of views over a root view.
    • 译文: 使用导航堆栈在根视图上呈现一系列视图。
    • 释义: 显示，呈现

  4. indicates

    • 例句: A value-destination indicates that you are pushing a value onto the path.
    • 译文: "Value destination"这种用法表明/意味着你正在将一个值推送到导航路径（Navigation path）上。
    • 释义: 表明，意味着

快速启动（两步完成）​

第一步：启动服务器（终端 1）​

cd free-claude-code
uv run uvicorn server:app --host 0.0.0.0 --port 8082

✅ 成功标志： 看到 Uvicorn running on http://0.0.0.0:8082 提示 ⚠️ 注意： 不要关闭此窗口，必须保持运行状态

第二步：启动 Claude Code（终端 2）​

新开一个终端窗口，运行：

ANTHROPIC_AUTH_TOKEN="freecc" ANTHROPIC_BASE_URL="http://localhost:8082" claude

⚙️ 切换模型配置​

根据需求修改 .env 文件来切换底层模型：

切换为 Kimi K2（编程能力最强）

sed -i '' 's|MODEL="nvidia_nim/z-ai/glm4.7"|MODEL="nvidia_nim/moonshotai/kimi-k2-instruct"|' /Users/a1234/free-claude-code/.env

切换为 GLM-4.7（中文能力较强）

sed -i '' 's|MODEL="nvidia_nim/moonshotai/kimi-k2-instruct"|MODEL="nvidia_nim/z-ai/glm4.7"|' /Users/a1234/free-claude-code/.env

验证当前模型

grep MODEL /Users/a1234/free-claude-code/.env

🔄 应用更改​

修改模型后，需重启服务器才能生效：

  1. 在终端 1 按 Ctrl+C 停止当前服务
  2. 重新运行启动命令：
cd free-claude-code
uv run uvicorn server:app --host 0.0.0.0 --port 8082

📝 注意事项​

事项说明
🟢 保持运行终端 1（服务器）必须一直开着，关闭后 Claude Code 将无法连接
💡 显示说明底层使用 NVIDIA NIM 免费模型，界面显示 "Sonnet 4.6" 属正常现象
⏱️ 使用限额免费额度限制为 40 次 / 分钟

通用英语单词短语收集

2026年4月29日 · 阅读需 2 分钟

  1. Based on the principle of voluntariness

    • 例句: Membership in the union is based on the principle of voluntariness—no employee is forced to join.
    • 译文: 加入工会遵循自愿原则，没有任何员工被强迫参加。
    • 释义: 根据自愿的原则

  2. One for all, all for one

    • 例句: The three friends made a pact: one for all, all for one. They promised to stand by each other no matter what challenges lay ahead.
    • 译文: 三位朋友立下誓言：人人为我，我为人人。他们承诺无论未来面临何种挑战，都会相互扶持。
    • 释义: 人人为我，我为人人

  3. Be careful, though

    • 例句: The restaurant is amazing and cheap. Be careful, though: it gets really crowded on weekends.
    • 译文: 那家餐厅很棒，还很便宜。不过要注意：周末人会特别多。
    • 释义: 不过要注意（用于补充警示信息）

  4. embedded in

    • 例句: These attitudes are embedded in our society.
    • 译文: 这些观念根植于我们的社会之中。
    • 释义: 根植于，深深嵌入

  5. enthusiasts

    • 例句: She is one of the most dedicated fitness enthusiasts I have ever met.
    • 译文: 她是我见过的最投入的健身爱好者之一。
    • 释义: 爱好者，发烧友
